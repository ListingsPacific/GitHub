# Marine Listings Pacific

Deployable React + Vite starter for a South Pacific yacht marketplace.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

1. Upload this folder to GitHub.
2. In Netlify choose **Add new project**.
3. Choose **Import an existing project**.
4. Connect your GitHub repo.
5. Netlify should detect:
   - Build command: `npm run build`
   - Publish directory: `dist`
6. Click **Deploy site**.

## Notes

- Replace placeholder boats in `src/App.jsx` with your real listings.
- Replace phone/email with your actual brokerage details.
- Replace the sample images with listing photos or brand assets.
